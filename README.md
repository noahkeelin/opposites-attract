# Opposites Attract

A fun project for the Scope Hackathon Fall 2022

## Frontend

Built with ReactJS and TailwindCSS

```sh
cd oa-frontend
npm i
npm start
```

Hosted locally on port 3000

## Backend

Built with NodeJS and ExpressJS

```sh
cd oa-backend
npm i
npm start
```

Hosted locally on port 5500
